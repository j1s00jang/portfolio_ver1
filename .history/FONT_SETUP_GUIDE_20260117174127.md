# Sofia Pro 폰트 설정 가이드

모든 사용자가 Sofia Pro 폰트를 볼 수 있도록 설정하는 방법입니다.

## 방법 1: Adobe Fonts 사용 (권장) ⭐

### 장점
- 간단하고 빠름
- 라이선스 문제 없음
- 자동으로 최적화됨

### 설정 방법

1. **Adobe Fonts 프로젝트 생성**
   - https://fonts.adobe.com 접속
   - 로그인 (Adobe 계정 필요)
   - "Add fonts" 클릭
   - "Sofia Pro" 검색 후 추가
   - "Web project" 생성

2. **프로젝트 ID 확인**
   - 생성된 프로젝트에서 링크 확인
   - 예: `https://use.typekit.net/abc123xyz.css`
   - 프로젝트 ID: `abc123xyz`

3. **index.html에 링크 추가**
   ```html
   <link rel="stylesheet" href="https://use.typekit.net/YOUR_PROJECT_ID.css" />
   ```
   - `YOUR_PROJECT_ID`를 실제 프로젝트 ID로 교체
   - 현재 `index.html`에 주석 처리된 부분을 찾아서 주석 해제하고 ID 입력

## 방법 2: 폰트 파일 직접 호스팅

### 장점
- Adobe 계정 불필요
- 완전한 제어 가능

### 설정 방법

1. **폰트 파일 준비**
   - Sofia Pro 폰트 파일 필요 (.woff2 또는 .woff 형식)
   - 폰트 파일을 `src/assets/fonts/` 폴더에 넣기
   - 예:
     ```
     src/assets/fonts/
       ├── SofiaPro-Regular.woff2
       ├── SofiaPro-Medium.woff2
       └── SofiaPro-Bold.woff2
     ```

2. **@font-face 선언 활성화**
   - `src/index.css` 파일 열기
   - 파일 상단의 주석 처리된 `@font-face` 부분을 찾기
   - 주석 해제 (`/* */` 제거)
   - 파일 경로를 실제 파일명에 맞게 수정

3. **폰트 파일 경로 확인**
   - Vite에서는 `./assets/fonts/` 경로 사용
   - 파일명이 다르면 경로 수정

### 주의사항
- 폰트 파일의 라이선스 확인 필요
- 상업적 사용 시 라이선스 구매 필요할 수 있음

## 현재 상태

현재는 폰트가 로드되지 않아서:
- ✅ **Chrome**: 사용자 컴퓨터에 설치된 Sofia Pro가 있으면 보임
- ❌ **Safari**: 기본 폰트로 표시됨 (Safari는 로컬 폰트 사용 안 함)
- ❌ **다른 사용자**: 컴퓨터에 폰트가 없으면 기본 폰트로 표시됨

위의 방법 중 하나를 선택하여 설정하면 모든 사용자가 Sofia Pro 폰트를 볼 수 있습니다.
