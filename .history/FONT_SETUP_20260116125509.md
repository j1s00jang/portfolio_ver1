# Sofia Pro 폰트 설정 가이드

Sofia Pro 폰트를 사용하는 방법은 두 가지가 있습니다:

## 방법 1: 폰트 파일 직접 사용 (권장)

1. Sofia Pro 폰트 파일을 다운로드합니다 (`.woff2` 또는 `.woff` 형식)
2. `src/assets/fonts/` 폴더를 생성하고 폰트 파일을 넣습니다
3. `src/index.css` 파일에서 주석 처리된 `@font-face` 부분을 해제합니다
4. 폰트 파일 경로를 실제 파일명에 맞게 수정합니다

예시:
```
src/assets/fonts/
  ├── SofiaPro-Regular.woff2
  ├── SofiaPro-Medium.woff2
  └── SofiaPro-Bold.woff2
```

## 방법 2: Adobe Fonts 사용

1. Adobe Fonts (fonts.adobe.com)에서 Sofia Pro를 프로젝트에 추가합니다
2. Adobe Fonts에서 제공하는 프로젝트 ID를 받습니다
3. `index.html` 파일의 `<head>` 섹션에서 주석 처리된 link 태그를 해제합니다
4. `YOUR_PROJECT_ID`를 실제 프로젝트 ID로 교체합니다

예시:
```html
<link rel="stylesheet" href="https://use.typekit.net/abc123xyz.css">
```

## 현재 상태

현재 CSS에는 'Sofia Pro' 폰트가 기본 폰트로 설정되어 있습니다. 
위의 방법 중 하나를 선택하여 폰트를 로드하면 자동으로 적용됩니다.
